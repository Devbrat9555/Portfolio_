Personal Portfolio Website - Devbrat Yadav

This is my personal portfolio website where I showcase my projects, skills, and experience as a Full-Stack Developer and DSA enthusiast.

Tech Stack:
⚛ React.js | ⚡ Vite | 🎨 Tailwind CSS | 🌐 JavaScript / JSX

Features:
- Responsive and modern UI with Emerald/Black theme
- Projects showcase (AKTU Prep, Fixit, etc.)
- DSA milestones and About Me section
- Smooth animations using Framer Motion & GSAP

Installation & Setup:
# Clone the repository
git clone https://github.com/Devbrat9555/Portfolio_

# Navigate to project folder
cd my-portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
